# Finalization Rules

## Validate the build after doing work

- [ ] Always run `npm run lint` to validate code style and catch unused variables
- [ ] Always run `npm run typecheck` to validate TypeScript types and catch type errors
- [ ] Always run `npm run build` to validate the new code works on the app
- [ ] Always run `npm run test` to validate all tests pass

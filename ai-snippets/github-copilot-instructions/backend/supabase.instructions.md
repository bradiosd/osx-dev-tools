# Supabase Rules

## Steps to taking when dealing with Supabase

- [ ] Assume Supabase is always running locally
- [ ] When creating new migration files always create new ones and don't modify old ones
- [ ] When making database changes always update the types in the api or webapp
- [ ] Never stop the Supabase instance running locally
